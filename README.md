# Quantum Computing — Interactive Notes
 
*Inspired by the Quantum Computing course of Prof. Marco Chiani and the notes of **Lorenzo Valentini**, Università di Bologna*

Self-contained interactive notes covering linear algebra, quantum mechanics, entanglement and quantum circuits — with live visualisations, interactive canvases and a drag-and-drop circuit simulator. No installation, no build step, runs entirely in the browser.

🌐 **Live site:** [https://Fre0Grella.github.io/quantum-notes/](https://github.com/Fre0Grella/quantum-notes*)
